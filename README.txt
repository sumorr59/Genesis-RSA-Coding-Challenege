How to build the app:

1. Dependencies:
    - rsa module --> pip install rsa
    - sslib module for sharding --> pip install sslib

2. Running the program
    - Open a terminal window to the directory where the main.py file is located
    - Run the python file using --> python main.py
    - Enter the message you would like to encrypt and decrypt
    - Enter the number of shards you want the private key to be split into and the number of shards you want it to
    take to recover

3. Running the unit test
    - Open a terminal window to the directory where the unittest.py file is located
    - Run the unitTest file using --> python ShamirUnitTest.py
